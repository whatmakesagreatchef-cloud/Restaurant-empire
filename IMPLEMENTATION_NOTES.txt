üîß HOW IT WORKS
==========================================

TUTORIAL TRIGGER FLOW:
1. Player takes action (e.g., opens first venue)
2. shouldShowTutorial() checks if tutorial needed
3. Tutorial modal displays if conditions met
4. Player progresses through steps
5. Completion tracked in state.tutorialDone

HINT SYSTEM FLOW:
1. Each week, check venue/state conditions
2. getActiveHints() returns relevant hints
3. Display hints as alert boxes in UI
4. Player can dismiss or act on hints
5. Hints reappear if problem persists

MILESTONE FLOW:
1. After each week, checkMilestones() runs
2. Compare state against milestone conditions
3. If met, show celebration modal
4. Apply rewards (cash, reputation, prestige)
5. Track in state.milestonesAchieved

TOOLTIP FLOW:
1. UI renders with tooltip() wrapper
2. Player taps "?" icon
3. Tooltip content displays from TOOLTIPS object
4. Works on any metric/section
